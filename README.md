## 📌 Descripción

Este repositorio contiene las tareas, desafíos y proyectos desarrollados durante el curso:

> **Backend con Node y Express – Generación G99**

El objetivo del curso es aprender a construir aplicaciones backend utilizando tecnologías modernas del ecosistema JavaScript, incluyendo la creación de APIs, manejo de bases de datos, autenticación y buenas prácticas de desarrollo.

---

## 🛠 Tecnologías utilizadas

Durante el curso se trabajará con:

- Node.js  
- Express.js  
- npm  
- JSON  
- APIs REST  
- Middlewares  
- Manejo de archivos con `fs`  
- PostgreSQL / SQL (según módulo)  
- Autenticación con JWT  
- Buenas prácticas de arquitectura backend
